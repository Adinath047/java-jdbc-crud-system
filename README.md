# Student Management System (JDBC)

A console-based backend application built using Java and JDBC that manages student records with database integration.

## Features
- Student Registration
- Login Authentication
- Insert / View / Update / Delete (CRUD)
- MySQL Database Connectivity
- PreparedStatement for SQL safety
- Menu-driven CLI interface

## Tech Stack
- Java
- JDBC
- MySQL
- SQL
